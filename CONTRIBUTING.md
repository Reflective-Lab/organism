# Contributing to Organism

Thanks for your interest in contributing to Organism.

## Getting Started

1. Fork the repository
2. Create a feature branch from `main`
3. Make your changes
4. Run `just lint` to ensure code quality
5. Submit a pull request

## Development

- Rust edition 2024, minimum rust-version 1.94
- `unsafe` code is forbidden
- Clippy pedantic is enforced

## Pull Requests

- Keep PRs focused — one logical change per PR
- Write clear commit messages
- Ensure all checks pass before requesting review

## Code of Conduct

Be respectful, constructive, and collaborative. We reserve the right to remove contributions or ban participants who do not meet this standard.
